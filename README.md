# 反馈助手 (Feedback Assistant)

基于 macOS 环境构建的语文课后学习反馈生成助手，用于帮助教师根据“课堂录音”或“课堂逐字稿”快速生成给家长的课后学习反馈。

## 🌟 核心特色
1. **WPS 纯文本兼容**：输出结果绝对不包含任何 Markdown 标记（如 `*` 或 `#`）和数字引用角标（如 `[1]`），可直接复制发送给家长。
2. **macOS 录音直调**：后端提供本地语音备忘录自动读取服务（需授权）以及专用的本地 `recordings` 拖拽目录。
3. **安全拖拽导入**：支持从 macOS 语音备忘录应用中直接拖拽音频卡片到浏览器界面进行转写生成。
4. **教研规则严格执行**：学生主动提问自动提为第一句、文言考点自动分类归纳、其余文本描述控制在 3-5 句。

---

## 🛠️ 项目结构
- `docs/`
  - `roles/`：系统角色定义（总控、前端、后端、UI设计师）
  - `pitfalls_rules.md`：开发避坑规则库 (实时更新)
  - `prompt.md`：调优后的系统大模型提示词
- `backend/`：本地 Node.js (Express) 服务端
- `frontend/`：Vite + React 网页前端
- `recordings/`：本地音频存放目录

## 🚀 启动与使用指南

### 第一步：启动后端服务
1. 打开终端并进入 `backend` 目录：
   ```bash
   cd "/Users/ziwelz/工作/AI/反馈助手/backend"
   ```
2. 安装依赖并启动（若已安装直接运行）：
   ```bash
   npm run start
   ```
   *后端将运行在 http://localhost:5001*

### 第二步：启动前端界面
1. 打开新的终端并进入 `frontend` 目录：
   ```bash
   cd "/Users/ziwelz/工作/AI/反馈助手/frontend"
   ```
2. 启动 Vite 开发服务：
   ```bash
   npm run dev
   ```
   *前端开发环境将运行在 http://localhost:5173*

### 第三步：使用反馈助手
1. 在浏览器中打开 http://localhost:5173
2. 点击右上角 **配置 API Key**，输入您的 Gemini API 密钥并保存。
3. **使用录音文件**：
   - 可以直接从 macOS 语音备忘录软件中，拖拽需要的录音到“拖拽区域”中。
   - 或者，在左下角“最近 macOS 语音备忘录”中，直接点击需要的录音进行处理。
4. **使用文字稿**：
   - 切换到“课堂文字稿分析”标签。
   - 粘贴您的课堂逐字稿内容，点击“生成课后学习反馈”。
5. 生成完成后，点击 **复制纯文本** 按钮即可一键获取 WPS 兼容的反馈信息。
